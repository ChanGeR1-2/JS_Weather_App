# JS_Weather_App
This project has been completed to learn and practise using asynchronous Javascript functions, the fetch API and integrating a client-side application with external APIs and processing the information.

### Functionality
- Use of the Geolocation API to get the user's location and automatically display the weather for their area.
- The user can enter full or partial addresses to get information.
- Information is displayed hourly for the current day and overviews are displayed for the following week.
- Users can change the units to be displayed.
- Unique weather symbols.
- Wind direction accurately represented.
- Responsive UI for mobile.
